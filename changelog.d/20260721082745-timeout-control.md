Add `TimeoutControl` for cooperative timeout cancellation.

`timeout()` now passes `{control}` to the callback. Destructure the `TimeoutControl` from the argument object:

```js
timeout({timeout: 5000}, ({control}) => {
  control.check()           // throws TimeoutError between async operations
  fetch(url, {signal: control.signal})  // cancel long I/O
})
```

Available properties:

- `signal` — AbortSignal aborted when the timeout fires (pass to `fetch()`, etc.)
- `check()` — throws `TimeoutError` between async operations to bail early
- `timedOut` — boolean
- `remaining()` — milliseconds until deadline

Backward compatible: callbacks that ignore the parameter continue to work.
